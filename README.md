# DeepLearningCNN-BasedGroundContactDetectionDuringSprintAcceleration
This repository contains the code for the 1D CNN used in the scientific paper Deep Learning CNN based ground contact detection during sprint acceleration

# Input Data Structure
athlete	run	position	binary	ares	gres

binary = label, ares & gres = features
athlete, run, position = groups. Used for train, val, test split
